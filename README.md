# FloodSOS-Complete

> **Lưu ý:** Toàn bộ tài liệu hướng dẫn cài đặt, cấu trúc kiến trúc (Architecture), hướng dẫn chạy bằng Docker, và Unit Testing đã được gộp chung vào file `README.md` ở thư mục gốc của dự án để đảm bảo tính đồng bộ.

Vui lòng xem chi tiết tại: [Root README.md](../../README.md)

---

Thư mục này chứa:
1. `frontend-flutter/`: Source code ứng dụng di động Flutter (Giao diện Calm Crisis). Tích hợp `.env` config.
2. `Sos-backend/`: Source code server Node.js. Đã tổ chức theo mô hình MVC, chạy ở port 3002, tích hợp Jest Unit tests.
